# 第三方 GitHub 提示词仓库

XSVO `v0.8.6` 会把原作者接入过的提示词源采集到管理员公共提示词库中。采集时只保留可显示的远程图片 URL，不会把封面图写入浏览器本地素材存储。

| 地址                                                               | 状态                       |
| ------------------------------------------------------------------ | -------------------------- |
| https://github.com/EvoLinkAI/awesome-gpt-image-2-API-and-Prompts   | 当前仓库不可访问，自动跳过 |
| https://github.com/ZeroLu/awesome-gpt-image                        | 已采集                     |
| https://github.com/ImgEdify/Awesome-GPT4o-Image-Prompts            | 已采集                     |
| https://github.com/YouMind-OpenLab/awesome-gpt-image-2             | 已采集                     |
| https://github.com/YouMind-OpenLab/awesome-nano-banana-pro-prompts | 已采集                     |
| https://github.com/davidwuw0811-boop/awesome-gpt-image2-prompts    | 已采集                     |

当前验证可采集约 874 条带可显示远程封面的提示词；`img.shields.io` 徽章、README 头图和统计图会被过滤。
